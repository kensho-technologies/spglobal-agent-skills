# S&P Capital IQ Excel Pro Skill


## Overview

This skill enables Claude to generate professional S&P Capital IQ Excel formulas and complete financial models with live data feeds. It provides comprehensive coverage of both modern SPG functions and legacy CIQ functions.

## What This Skill Does

- **Generate Capital IQ formulas** for any financial datapoint
- **Create professional Excel financial models** with proper formatting and color coding
- **Build valuation models** (DCF, LBO, merger models)
- **Construct peer analysis tables** with live comparable company data
- **Pull market data** (prices, volumes, returns, beta)
- **Access macroeconomic data** (treasury rates, FX pairs)
- **Retrieve consensus estimates** (Visible Alpha integration)
- **Extract officer information** (CEO, CFO names)

## Example Use Cases

1. **DCF Valuation Model** - Pull historical financials, beta, rates; build complete DCF
2. **Peer Comparison** - Build comp table with multiples, growth rates, margins
3. **Consensus Analysis** - Track analyst estimates across peer group
4. **Market Dashboard** - Stock prices, returns, beta, CAPM analysis
5. **Rates & FX** - Treasury yield curves, currency matrices
6. **Credit Analysis** - Ratings, debt schedules, coverage ratios

## Requirements

- S&P Capital IQ Excel Plugin installed
- Valid Capital IQ subscription
- Excel (formulas don't work in Google Sheets or LibreOffice)
- Appropriate data entitlements for premium features
